Introduction

Track Internet provides a simple application to track and log the state of the device's internet connection

Requirements

Mandatory Requirements

For the application to run:

  • wifi must be enabled
  • SD card inserted
  • Permission to access the device's location

Other Requirements

To allow parts of the app to work other fields have to be populated

  • Too track and record the state of the internet connection a log file name has to be provided
  • Too email the log file an email address has to be provided.

Operation

Startup

Without wifi enabled or an sd card the app will not run and will exit immediately with an error message. On startup for the first time the app prompts with "Allow Track Internet to access this device's location" If "DENY" is selected, the app immediately exits. If the app is started again the same prompt will appear, ticking the "Don't ask again" check box is ignored, as the app won't run until the "ALLOW" option is selected. Once all the starting requirements are satisfied, the app will present the main screen allowing an email address and a log file name to be entered. The email address and log file name will be remembered, so there is no need to enter these each time the app is started.

Running

Once running, by tapping the appropriate button the app will:

  • record outages in the log file
  • stop recording outages in the log file
  • display outages which have been recorded in the log file
  • email the log file to the entered email address
  • scan for and display wifi networks

Recording

When record outages is selected, the app prompts if the existing log file is to be deleted or not. Each time the recording of outages is stopped or started a message is written to the log file. When recording is started, the model of the device and the android software version is also written to the log file. The app records changes to the internet connection by writing the date/time and the state of the internet connection to the log file. For the state of the connection, the app writes "internet connected" or "internet disconnected". If the state changes to "connected", the app writes the type of network connection, typically wifi or mobile data. In the case of a wifi connection, the app also records the SSID (wifi network name) and the frequency of the wifi network.

Menu

Via the three dots at the top right hand side of the display that can be made to:

  • stop recording and quit
  • keep recording and quit
  • get help, display this information
