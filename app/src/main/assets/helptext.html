Introduction

Track Internet provides a simple application to track and log the state of the device's internet connection

Requirements

Mandatory Requirements

For the application to run:

  • wifi must be enabled
  • SD card inserted
  • Permission to access the device's location

Other Requirements

To allow parts of the app to work other fields have to be populated

  • Too track and record the state of the internet connection a log file name has to be provided
  • Too email the log file an an email address has to be provided.

Operation

Without wifi enabled or an sd card the app will not run and will exit immediately with an error message. On startup for the first time the user is prompted with "Allow Track Internet to access this device's location" If the user selects "DENY", the app immediately exits. If the app is started again the same prompt will appear, ticking the "Don't ask again" check box is ignored. The app won't run until the "ALLOW" option is selected. Once all the starting requirements are satisfied, the app will present the main screen allowing an email address and a log file name to be entered. The email address and log file name will be remembered ach time the app is restarted.
